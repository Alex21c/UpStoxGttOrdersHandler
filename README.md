# UpStoxGttOrdersHandler

place and cancels GTT orders for upstox

## Backend .env file

```javascript
API_KEY=get from upstox
API_SECRET=get from upstox
REDIRECT_URL=http://localhost:3000/callback
FRONTEND_URL=http://localhost:4000
```

## install frontEnd and Backend packages

```javascript
cd frontEnd
npm install

cd backEnd
npm install
```

## Create a dist of frontEnd using build command

```javascript
npm run build
```

## Install npm serve package to access frontend static files on localhost server

```javascript
npm install -g serve
```

## To Launch forntEnd Server using npm serve package

```bash
frontEnd/launchFrontEnd.bat
```

## To launch backend node server

```bash
backEnd/launchBackEnd.bat
```
